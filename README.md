# Projet JavaScript : Système de Gestion de Tâches (TaskFlow)
## Parcours Génie Informatique - L3

Bienvenue dans votre projet de fin de module JavaScript. Ce projet a pour but de vous familiariser avec les frameworks frontend modernes (React ou Vue.js) tout en appliquant les bonnes pratiques de développement.

### Objectif du Projet
Développer une application de gestion de tâches ("To-Do List") avancée permettant de gérer son emploi du temps de manière interactive.

### Fonctionnalités Attendues (MVP)
1. Gestion des Tâches : Ajouter, modifier, supprimer et cocher des tâches comme terminées.
2. Catégories & Priorités : Assigner une catégorie (Travail, Personnel, Études) et un niveau de priorité (Basse, Moyenne, Haute) à chaque tâche.
3. Filtrage & Recherche : Pouvoir filtrer les tâches par état (Toutes / En cours / Terminées) et par catégorie.
4. Persistance des Données : Utilisation du LocalStorage pour sauvegarder les données entre les sessions.
5. Interface Responsive : Un design propre et utilisable sur mobile et desktop.

### Choix du Framework
Vous avez la liberté de choisir entre :
- React (avec Hooks) : Pour une approche basée sur les fonctions et un écosystème vaste.
- Vue.js (Composition API) : Pour une approche intuitive avec des templates et une réactivité puissante.

### Structure du Dépôt
- starters/ : Contient les bases de code pour démarrer rapidement.
- glossaire.md : Guide des termes techniques essentiels.

### Consignes de Rendu
1. Le code doit être versionné avec Git.
2. Un fichier INSTALL.md doit expliquer comment lancer votre projet.
3. Bonus : Utilisation de bibliothèques de composants (TailwindCSS, Bootstrap, Vuetify, MUI, etc.).
