# Glossaire du Projet (L3 Génie Informatique)

Ce glossaire contient les termes essentiels pour comprendre le développement moderne avec JavaScript, React et Vue.js.

## Concepts JavaScript (ES6+)
- DOM (Document Object Model) : Représentation structurée du HTML sous forme d'arbre utilisable par JavaScript.
- ES6 / ECMAScript : La norme moderne de JavaScript (incluant let, const, les fonctions fléchées, etc.).
- Asynchrone (async/await) : Permet d'exécuter du code sans bloquer le thread principal, essentiel pour les appels API.
- Destructuring : Syntaxe permettant d'extraire des données de tableaux ou d'objets dans des variables distinctes.
- Module (import/export) : Système permettant de diviser le code en plusieurs fichiers réutilisables.

## Concepts Framework (React & Vue)
- Composant : Bloc de construction réutilisable de l'interface utilisateur.
- Props : Données passées d'un composant parent à un composant enfant.
- État (State) : Données internes d'un composant qui, lorsqu'elles changent, déclenchent une mise à jour de l'affichage.
- Virtual DOM : Copie légère du DOM réel utilisée par les frameworks pour optimiser les performances de rendu.
- Hooks (React) / Composition API (Vue) : Fonctions permettant d'utiliser l'état et d'autres fonctionnalités sans écrire de classes.
- SPA (Single Page Application) : Application web qui ne recharge qu'une seule page HTML et met à jour dynamiquement le contenu.

## Outils de Développement
- NPM / PNPM / Yarn : Gestionnaires de paquets pour installer des bibliothèques tierces.
- Vite : Outil de build moderne et rapide pour le développement frontend.
- JSX / Templates : Syntaxe permettant d'écrire le rendu visuel (HTML-like) directement dans le code JavaScript.
